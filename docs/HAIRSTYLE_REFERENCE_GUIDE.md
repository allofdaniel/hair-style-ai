# 헤어스타일 레퍼런스 가이드

AI 헤어스타일 생성을 위한 상세 스타일 설명서입니다.

---

## 남성 헤어스타일 (Male Styles)

### 다운펌 (Down Perm)

#### 1. 쉼표머리 (Comma Hair)
- **설명**: 앞머리가 이마에 자연스럽게 떨어지면서 쉼표(,) 모양을 형성하는 스타일
- **특징**:
  - 앞머리 끝이 부드럽게 한쪽으로 휘어짐
  - 이마의 70-80%를 가리는 길이
  - 옆머리는 깔끔하게 정돈
  - 뒷머리는 자연스러운 볼륨
- **적합한 얼굴형**: 긴 얼굴, 각진 얼굴
- **연예인 예시**: 박보검, 차은우, BTS V

#### 2. 가일펌 (Gail Perm)
- **설명**: 부드러운 S자 웨이브가 아래로 자연스럽게 떨어지는 펌
- **특징**:
  - 중간 길이의 앞머리
  - 부드럽고 자연스러운 웨이브
  - 광택감 있는 마무리
  - 볼륨감이 살아있음
- **적합한 얼굴형**: 둥근 얼굴, 달걀형 얼굴
- **연예인 예시**: 정해인, 송강

#### 3. 포마드 다운펌 (Pomade Down Perm)
- **설명**: 머리를 뒤로 깔끔하게 넘긴 슬릭백 스타일
- **특징**:
  - 이마가 완전히 드러남
  - 젖은 듯한 광택 마무리
  - 단정하고 깔끔한 인상
  - 옆 가르마 또는 뒤로 빗어 넘김
- **적합한 얼굴형**: 달걀형, 하트형 얼굴
- **연예인 예시**: 현빈, 조인성

#### 4. 내추럴 다운 (Natural Down)
- **설명**: 인위적이지 않게 자연스럽게 내린 스타일
- **특징**:
  - 드라이만으로도 스타일링 가능
  - 부드러운 질감
  - 자연스러운 가르마
  - 편안하고 캐주얼한 느낌
- **적합한 얼굴형**: 모든 얼굴형
- **연예인 예시**: 공유, 이동욱

### 투블럭 (Two Block)

#### 5. 댄디컷 (Dandy Cut)
- **설명**: 옆과 뒤는 짧게, 위는 볼륨있게 유지하는 기본 투블럭
- **특징**:
  - 옆머리 3-6mm로 짧게 컷
  - 윗머리는 5-8cm 길이
  - 깔끔하고 단정한 인상
  - 비즈니스 캐주얼에 적합
- **적합한 얼굴형**: 둥근 얼굴, 사각형 얼굴
- **연예인 예시**: 이종석, 박서준

#### 6. 언더컷 (Undercut)
- **설명**: 옆면을 확실하게 밀고 윗머리와의 경계를 두는 스타일
- **특징**:
  - 옆머리 0-3mm 매우 짧게
  - 윗머리와 확실한 분리감
  - 엣지있고 트렌디한 느낌
  - 스타일링에 따라 다양한 연출 가능
- **적합한 얼굴형**: 긴 얼굴, 달걀형 얼굴
- **연예인 예시**: 지드래곤, BTS 정국

#### 7. 투블럭 기본 (Two Block Basic)
- **설명**: 클래식한 투블럭의 정석 스타일
- **특징**:
  - 옆머리와 윗머리의 자연스러운 연결
  - 앞머리를 내리거나 올릴 수 있음
  - 관리가 쉬움
  - 다양한 변형 가능
- **적합한 얼굴형**: 모든 얼굴형
- **연예인 예시**: 이민호, 김수현

#### 8. 모히칸 투블럭 (Mohawk Two Block)
- **설명**: 중앙에 볼륨을 집중시킨 모히칸 스타일
- **특징**:
  - 정수리 중앙에 높이감
  - 양 옆은 짧게 밀어줌
  - 과감하고 개성있는 인상
  - 스타일링 제품 필수
- **적합한 얼굴형**: 둥근 얼굴 (세로 길이 보완)
- **연예인 예시**: 빅뱅 탑, EXO 카이

### 펌 (Perm)

#### 9. 애즈펌 (Ash Perm)
- **설명**: 자연스러운 S자 커브가 전체적으로 들어간 펌
- **특징**:
  - 부드러운 웨이브
  - 볼륨감 UP
  - 세련된 분위기
  - 드라이만으로 스타일링
- **적합한 얼굴형**: 각진 얼굴, 긴 얼굴
- **연예인 예시**: 이민호, 김수현

#### 10. 가르마펌 (Garma Perm)
- **설명**: 가르마를 중심으로 양쪽에 웨이브를 넣은 펌
- **특징**:
  - 6:4 또는 7:3 가르마
  - 양쪽으로 자연스럽게 흐르는 웨이브
  - 지적이고 세련된 이미지
  - 중간 길이 이상에 적합
- **적합한 얼굴형**: 달걀형, 하트형 얼굴
- **연예인 예시**: 송중기, 남주혁

#### 11. 리젠트펌 (Regent Perm)
- **설명**: 뒤로 넘기면서 볼륨감 있게 연출하는 펌
- **특징**:
  - 앞머리를 뒤로 넘김
  - 옆에서 봤을 때 C자 커브
  - 포멀하고 세련된 느낌
  - 턱시도/정장과 어울림
- **적합한 얼굴형**: 사각형, 긴 얼굴
- **연예인 예시**: 원빈, 장동건

#### 12. S컬펌 (S Curl Perm)
- **설명**: 선명한 S자 웨이브가 전체적으로 들어간 펌
- **특징**:
  - 뚜렷한 S자 커브
  - 로맨틱한 분위기
  - 중간~긴 길이에 적합
  - 예술적이고 감성적인 이미지
- **적합한 얼굴형**: 각진 얼굴
- **연예인 예시**: 이준기, 김재욱

#### 13. 히피펌 (Hippie Perm)
- **설명**: 자유로운 느낌의 큰 웨이브 펌
- **특징**:
  - 루즈한 큰 웨이브
  - 보헤미안 스타일
  - 자연스럽고 편안한 느낌
  - 긴 머리에 최적
- **적합한 얼굴형**: 모든 얼굴형
- **연예인 예시**: 정용화, 이홍기

### 장발 (Long Hair)

#### 14. 울프컷 (Wolf Cut)
- **설명**: 레이어드가 많이 들어간 샤기한 스타일
- **특징**:
  - 뒤쪽이 길고 앞쪽은 짧음 (멀렛 변형)
  - 많은 층
  - 텍스처드한 질감
  - 트렌디하고 개성있음
- **적합한 얼굴형**: 둥근 얼굴
- **연예인 예시**: BTS 뷔, Stray Kids 현진

#### 15. 롱 레이어드 (Long Layered)
- **설명**: 어깨 길이의 레이어드 컷
- **특징**:
  - 자연스러운 층
  - 움직임과 볼륨
  - 아티스틱한 느낌
  - 다양한 스타일링 가능
- **적합한 얼굴형**: 사각형, 긴 얼굴
- **연예인 예시**: 장근석, 김희철

#### 16. 맨번 (Man Bun)
- **설명**: 뒤로 묶은 번 스타일
- **특징**:
  - 충분한 길이 필요 (최소 15cm)
  - 정수리에 번으로 묶음
  - 깔끔하면서도 트렌디
  - 언더컷과 조합 가능
- **적합한 얼굴형**: 모든 얼굴형
- **연예인 예시**: 강동원, 이진욱

### 숏 (Short)

#### 17. 크루컷 (Crew Cut)
- **설명**: 군인 스타일의 짧은 머리
- **특징**:
  - 전체적으로 짧음
  - 옆면 더 짧게
  - 깔끔하고 관리 쉬움
  - 운동선수 스타일
- **적합한 얼굴형**: 달걀형, 하트형
- **연예인 예시**: 현빈, 손흥민

#### 18. 아이비리그 (Ivy League)
- **설명**: 깔끔하고 단정한 클래식 스타일
- **특징**:
  - 옆 가르마
  - 깔끔하게 빗어 넘김
  - 프로페셔널한 이미지
  - 비즈니스에 적합
- **적합한 얼굴형**: 모든 얼굴형
- **연예인 예시**: 유재석, 강호동

#### 19. 버즈컷 (Buzz Cut)
- **설명**: 전체적으로 짧게 민 스타일
- **특징**:
  - 균일한 짧은 길이
  - 두상이 드러남
  - 매우 간편한 관리
  - 대담하고 미니멀
- **적합한 얼굴형**: 달걀형 (두상이 예쁜 경우)
- **연예인 예시**: 유아인, 하정우

### 페이드 (Fade)

#### 20. 로우 페이드 (Low Fade)
- **설명**: 귀 아래에서 시작하는 낮은 페이드
- **특징**:
  - 귀 아래부터 자연스러운 그라데이션
  - 보수적이면서도 세련됨
  - 다양한 스타일과 조합 가능
- **적합한 얼굴형**: 모든 얼굴형
- **연예인 예시**: 박재범, Crush

#### 21. 미드 페이드 (Mid Fade)
- **설명**: 중간 높이에서 시작하는 페이드
- **특징**:
  - 관자놀이 높이에서 시작
  - 가장 일반적인 페이드
  - 현대적이고 깔끔함
- **적합한 얼굴형**: 모든 얼굴형
- **연예인 예시**: 사이먼 도미닉, Gray

#### 22. 하이 페이드 (High Fade)
- **설명**: 높은 위치에서 시작하는 강한 페이드
- **특징**:
  - 정수리 가까이에서 시작
  - 대담하고 강렬한 인상
  - 스트릿/힙합 스타일
- **적합한 얼굴형**: 둥근 얼굴, 사각형 얼굴
- **연예인 예시**: 빈지노, Zico

#### 23. 스킨 페이드 (Skin Fade)
- **설명**: 피부가 보일 정도로 깔끔한 페이드
- **특징**:
  - 피부까지 자연스럽게 연결
  - 매우 샤프한 인상
  - 유지 관리 주기적 필요
- **적합한 얼굴형**: 달걀형, 하트형
- **연예인 예시**: 로꼬, BewhY

## 여성 헤어스타일 (Female Styles)

### 숏컷 (Short Cut)

#### 1. 픽시컷 (Pixie Cut)
- **설명**: 귀여운 요정 같은 짧은 머리
- **특징**:
  - 전체적으로 짧음
  - 윗머리에 텍스처
  - 귀가 드러남
  - 세련되고 시크함
- **적합한 얼굴형**: 달걀형, 하트형
- **연예인 예시**: 고준희, 한예슬

#### 2. 보브컷 (Bob Cut)
- **설명**: 턱선에서 깔끔하게 자른 단발
- **특징**:
  - 턱선 길이
  - 끝이 한 라인으로 정돈
  - 깔끔하고 우아함
  - 앞머리 유무 선택 가능
- **적합한 얼굴형**: 긴 얼굴, 각진 얼굴
- **연예인 예시**: 고윤정, 수지

#### 3. 허쉬컷 (Hush Cut)
- **설명**: 레이어드가 많이 들어간 숏컷
- **특징**:
  - 얼굴 주변에 레이어
  - 가벼운 느낌
  - 볼륨감
  - 트렌디하고 현대적
- **적합한 얼굴형**: 둥근 얼굴, 사각형 얼굴
- **연예인 예시**: 고윤정, 김고은

#### 4. 숏 울프컷 (Short Wolf Cut)
- **설명**: 짧은 버전의 울프컷
- **특징**:
  - 레이어와 텍스처가 많음
  - 약간 헝클어진 느낌
  - 개성있고 쿨한 이미지
- **적합한 얼굴형**: 모든 얼굴형
- **연예인 예시**: 박신혜, 한소희

### 앞머리 (Bangs)

#### 13. 시스루뱅 (See-through Bangs)
- **설명**: 가볍게 이마가 비치는 앞머리
- **특징**:
  - 얇은 가닥
  - 이마가 살짝 비침

## 레퍼런스 이미지 생성 가이드

각 스타일에 대한 AI 이미지 생성 시 다음 요소를 포함해야 합니다:

### 필수 요소
1. **정면/측면/뒷모습** 3각도 중 최소 1개
2. **머리카락 텍스처** 명확히 표현
3. **헤어라인** 자연스럽게 표현
4. **볼륨감과 실루엣** 강조
5. **조명** - 머리결이 잘 보이도록

### 권장 스타일
- 깔끔한 단색 배경 (흰색/회색 권장)
- 마네킹 또는 심플한 얼굴
- 머리에만 집중할 수 있도록
- 고해상도 (최소 512x512)
